h1. ShortWords

The two-letter-word list on your phone

h2. What is ShortWords?

ShortWords is an Android app that displays English two-letter-words and allows
you to filter by the starting letter of the word.

h2. Using ShortWords

Using ShortWords is simple. Tap the letter on the left and the list on the
right is filtered to only include words beginning with that letter. Tap the
letter again to see the complete list.
